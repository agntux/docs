# Installation

Follow these steps to install Agntux.

## Prerequisites

Before installing Agntux, make sure you have:

- Node.js 18 or higher
- npm or yarn package manager

## Installation Steps

1. Clone the repository
2. Install dependencies
3. Run the setup script

```bash
npm install agntux
```

## Verification

To verify your installation, run:

```bash
agntux --version
```
